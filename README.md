# **DigitalOrchestra**
an application developed for my master's thesis *(diplomarbete, diplomityö)*

## Requirements: 

- Sonic Pi (https://sonic-pi.net/) needs to be installed and running
- Python (https://www.python.org/)
- Install requirements into venv:
> `source venv/bin/activate`

> `pip install -r requirements.txt`

> `cd src`

> `python digital-orchestra.py`
